Bienvenido al archivo Readme para el ejercicio de Automatizacion E2E!
Este E2E fue desarrollado usando WebDriverIO con Cucumber en JavaScript.
A continuacion se presentan los pasos para la ejecucion de las pruebas.

Pre-requisitos:
1. Node.js minimo 16.15.1
2. Google Chrome version 122

Pasos:

1. Desde la CLI, navegar hacia el folder /Test Automation/E2E
2. Ejecutar el comando "npm install" para instalar las dependencias indicadas en el archivo package.json
3. Ejecutar el comando "npm run test" para iniciar las pruebas


Pasos opcionales:
1. Si deseas ver las pruebas en ejecucion, abre el archivo wdio.conf.js, en la linea 61 encontraras parametros usados al abrir el navegador, por defecto se incluye el modo headless
puedes comentar esa linea de codigo para que el navegador se abra normalmente y no headless


2. Si deseas ver a mas detalle el funcionamiento de webdriverIO durante al ejecucion de las pruebas puedes cambiar el nivel de logs desde el archivo wdio.conf.js en la linea 76;
En la linea 75 se indican los niveles que pueden ser utilizados!
